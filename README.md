# react-native-image-carousel

RN Image Carousel using Native

## Installation

```sh
npm install react-native-image-carousel
```

## Usage


```js
import { ImageCarouselView } from "react-native-image-carousel";

// ...

<ImageCarouselView color="tomato" />
```
